# sabi (name subject to change)

Ebook reader for Windows. Because all others suck (I'm sorry).

For now, it's made with Qt with C++. I couldn't get QML/QtQuick to look the way I wanted it to so it's QtWidgets. May end up switching to plain Win32 if I feel like it...

## (Totally not ambitious) Planned Features

- Snappy and light-feeling UI/UX
- Epub 2 and Epub 3 support
- Kepub format support
- Mobi + Azw3 format support
- CBZ and other comic format support
- Pointing to an OPDS online library
- Pointing to a local Calibre library 
- Pointing to a local or network/FTP folder
- Vertical, RTL, and CJK text support
- Ruby text support
- Paged and flow modes, page flip animations
- Somewhat customisable styling
- Sync with Goodreads
- Look up words in system dictionary (if it exists)
- Highlight and add notes to words/sentences
- Search for other cocurences of a word in book (with notes attached)
- Name/text replacement
- Basic TTS
- Fully accessible to screen readers regardless of TTS usage
- Sync between instances of the software across devices
